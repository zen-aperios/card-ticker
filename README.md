# card-ticker

Card ticker script based on your page-tested variant.

## Files

- `card-ticker.js`: source script
- `card-ticker.min.js`: minified production script
- `script.js`: mirror source for pen usage
- `HOW_TO_WEBFLOW.md`: mini setup guide

## Webflow CDN

```html
<script src="https://cdn.jsdelivr.net/gh/zen-aperios/card-ticker@main/card-ticker.min.js"></script>
```

## Required Markup

```html
<div class="ticker-slider" data-direction="left">
  <div class="track">
    <div>Card 1</div>
    <div>Card 2</div>
    <div>Card 3</div>
  </div>
</div>
```
